# 📝 MERN Todo App with Tailwind CSS

A beautiful, full-stack Todo application built with the MERN stack (MongoDB, Express.js, React, Node.js) and styled with Tailwind CSS.

![Todo App Demo](https://via.placeholder.com/800x400?text=Todo+App+Screenshot)

## ✨ Features

- ✅ Create, Read, Update, Delete todos
- 🎯 Mark todos as complete/incomplete
- 🔍 Filter todos (All, Pending, Completed)
- 📊 Real-time statistics (total, completed, pending)
- 🎨 Modern UI with Tailwind CSS
- 📱 Fully responsive design
- 🔔 Toast notifications for actions
- 💾 Persistent storage with MongoDB
- ⚡ Fast performance with Vite

## 🚀 Tech Stack

### Frontend
- React 18
- Vite
- Tailwind CSS
- Axios
- React Hot Toast
- Lucide React Icons

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- CORS
